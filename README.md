📊 World University Ranking Dashboard (2016–2023)

🔎 Project Overview
This project presents a comprehensive Power BI dashboard analyzing global university rankings, performance components, and international student trends from 2016 to 2023.

The dashboard explores:
1. Global university ranking trends
2. Teaching, research, and industry score components
3. Gender distribution patterns
4. International student growth
5. International outlook score analysis

🛠 Tools Used
Power BI, DAX, Data modeling, Data cleaning & transformation

📈 Key Insights
1. Consistent upward growth in international student numbers
2. Asia leading international expansion
3. Teaching and research scores strongly correlated with overall ranking stability
4. Score component drivers vary across institutions
